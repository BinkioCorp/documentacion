---
title: Change Password
description: Este endpoint permite a los usuarios cambiar su contraseña. El usuario debe proporcionar su contraseña actual y la nueva contraseña.
---

## POST `/api/auth/change-password`

### Solicitud

**Método:** POST  
**Ruta:** `/api/auth/change-password`

**Cuerpo de la solicitud:**

```json
{
  "currentPassword": "string",
  "newPassword": "string"
}

```
### Detalle de los Parámetros

| **Nombre**        | **Tipo**  | **Requerido** | **Descripción**                                                |
|-------------------|-----------|---------------|----------------------------------------------------------------|
| `currentPassword` | `string`  | Sí            | La contraseña actual del usuario.                              |
| `newPassword`     | `string`  | Sí            | La nueva contraseña que el usuario desea establecer.           |

### Respuestas

**200 - Contraseña cambiada exitosamente**

```json
{
  "message": "La contraseña ha sido cambiada exitosamente."
}

```

**400 - Contraseña actual incorrecta**

```json
{
  "message": "La contraseña actual es incorrecta."
}

```

**400 - Nueva contraseña no válida**

```json
{
  "message": "La nueva contraseña no cumple con los requisitos de seguridad."
}

```

**Notas:**

Si la contraseña actual es incorrecta, no se podrá cambiar la contraseña.

### Ejemplo de Uso

**Solicitud HTTP:**

```https
POST /api/auth/change-password HTTP/1.1
Content-Type: application/json

{
  "currentPassword": "MiContraseñaActual123",
  "newPassword": "MiNuevaContraseñaSegura456!"
}

```

**Respuesta Exitosa:**

```json
{
  "message": "La contraseña ha sido cambiada exitosamente."
}

```

**Respuesta con Error:**

```json
{
  "message": "La contraseña actual es incorrecta."
}

```
