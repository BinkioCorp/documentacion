---
title: Google Login
description: Este endpoint permite a los usuarios iniciar sesión utilizando un token generado por Google OAuth.
---

## POST `/api/auth/google-login`

### Solicitud

**Método:** POST  
**Ruta:** `/api/auth/google-login`

**Cuerpo de la solicitud:**

```json
{
  "token": "string"
}
```
### Detalle de los Parámetros

| **Nombre**        | **Tipo**  | **Requerido** | **Descripción**                                                |
|-------------------|-----------|---------------|----------------------------------------------------------------|
| `token`           | `string`  | Sí            | Token generado por Google para el usuario.                     |

### Respuestas

**200 - Inicio de sesión exitoso**

```json
{
  "message": "Inicio de sesión exitoso.",
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "user": {
    "id": "12345",
    "email": "usuario@dominio.com",
    "name": "Juan Pérez"
  }
}

```

**401 - Token inválido**

```json
{
  "message": "El token proporcionado es inválido o ha expirado."
}
```
**Notas:**

El token debe ser generado por Google OAuth y validado por el servidor.

### Ejemplo de Uso

**Solicitud HTTP:**

```https
POST /api/auth/google-login HTTP/1.1
Content-Type: application/json

{
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
}
```

**Respuesta Exitosa:**

```json
{
  "message": "Inicio de sesión exitoso.",
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "user": {
    "id": "12345",
    "email": "usuario@dominio.com",
    "name": "Juan Pérez"
  }
}
```

**Respuesta con Error:**

```json
{
  "message": "El token proporcionado es inválido o ha expirado."
}

```
