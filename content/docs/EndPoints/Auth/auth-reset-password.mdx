---
title: Reset Password
description: Este endpoint permite a los usuarios restablecer su contraseña utilizando un token proporcionado tras solicitar el restablecimiento de contraseña.
---

## POST `/api/auth/reset-password`

### Solicitud

**Método:** POST  
**Ruta:** `/api/auth/reset-password`

**Cuerpo de la solicitud:**

```json
{
  "token": "string",
  "newPassword": "string"
}
```
### Detalle de los Parámetros

| **Nombre**        | **Tipo**  | **Requerido** | **Descripción**                                                |
|-------------------|-----------|---------------|----------------------------------------------------------------|
| `token`           | `string`  | Sí            | Token de restablecimiento de contraseña enviado al usuario.    |
| `newPassword`     | `string`  | Sí            | Nueva contraseña que el usuario desea establecer.              |

### Respuestas

**200 - Contraseña restablecida exitosamente**

```json
{
  "message": "La contraseña se ha restablecido correctamente."
}

```

**400 - Error en la solicitud**

```json
{
  "message": "Token o nueva contraseña no proporcionados.",
  "errors": {
    "token": "Debe proporcionar un token válido.",
    "newPassword": "Debe proporcionar una nueva contraseña válida."
  }
}

```

**401 - Token inválido o expirado**

```json
{
  "message": "El token de restablecimiento es inválido o ha expirado."
}

```

**404 - Usuario no encontrado**

```json
{
  "message": "El token no corresponde a ningún usuario registrado."
}

```
**Notas:**

La nueva contraseña debe cumplir con los requisitos de seguridad establecidos, como longitud mínima y complejidad.

### Ejemplo de Uso

**Solicitud HTTP:**

```https
POST /api/auth/reset-password HTTP/1.1
Content-Type: application/json

{
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "newPassword": "NuevaContraseñaSegura123"
}

```

**Respuesta Exitosa:**

```json
{
  "message": "La contraseña se ha restablecido correctamente."
}

```

**Respuesta con Error:**

```json
{
  "message": "El token de restablecimiento es inválido o ha expirado."
}

```
